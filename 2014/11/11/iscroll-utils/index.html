iscroll-utils

发表于 2014-11-11 | 分类于 javascript

        在utils模块中通过自运行函数对外暴露一个me对象，通过me对象对外提供可调用的方法。

  • _prefixStyle方法：

    在模块最开始通过判断transform样式及其不同浏览器下的不同前缀样式是否在div的样式表中获取该浏览器的CSS前缀。之后_prefixStyle方法实现css前缀的添加。

  • extend方法：

    简单的对象属性扩展，没有实现深度扩展

  • prefixPointerEvent方法：

    修正PointerEvent，该样式具体功能可以参照Pointer Events,具体调用的地方及功能待补充

  • momentum方法：

    计算动量，该方法接受6个参数
    *current

    • start
    • time
    • lowerMargin
    • wrapperSize
    • deceleration

  • 扩展utils自身的属性(CSS支持情况)：

    • hasTransform 该浏览器是否支持transform样式
    • hasPerspective 该浏览器是否支持perspective样式
    • hasTouch 该浏览器是否支持touch事件
    • hasPointer IE10以上支持MSPointerEvent带前缀的事件类型
    • hasTransition 是否支持动画效果

  • isBadAnroid：

    检测所有低于build 535.19版本的android平台，该属性用于判断是否支持transitionDuration:0s,如果不支持则设置transitionDuration:0.001s,防止滚动条不滚动。

  • 扩展utils的style属性(CSS3.0动画效果相关的样式)：

    transform

  • hasClass方法

    该方法用于判断元素是否拥有某一class,该方法接受两个参数：

    • e dom元素
    • c 样式类名
      通过判断dom元素的className是否是以c的值开始或c的值之前存在空白符并且dom元素的className是否是以c的值为结尾或c的值之后存在空白符来判断该方法是否存在某一样式类

  • addClass

    空白

  • removeClass

    通过正则替换去掉给定的样式类名

  • offset

    空白

  • preventDefaultException

    空白
